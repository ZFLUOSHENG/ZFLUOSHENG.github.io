ArKTradingbot软件下载

下载地址？

✨我们的软件存放于阿里的钉钉群文件中，需要大家自行下载！

同时您也可以此时通过手机钉钉APP扫👇图，入群进行下载！

✨进入到钉钉群后，我们点击钉钉群右上角的文件夹按钮，你会看到以下几个文件：

✨第二步下载图中我标记的两个文件
1）ArkTradingbot-Beta.zip为软件本体
2）Data.zip为交易所商品历史数据，用于策略回测

打开软件

将您下载的文件存放于除C盘（系统盘）以外的任何盘符，最好在其它盘符里面新建一个文件夹重命名为：ArkTradingbot ;
将两个文件都放至于ArkTradingbot文件夹
步骤如下：
例：
盘符：D盘 👉 文件夹：./ArkTradingbot/ArkTradingbot-Beta/ArkTradingbot.exe
在D盘创建一个名为ArkTradingbot文件夹（除C盘外，任何盘符都可以）

将下载的文件剪切至ArkTradingbot文件夹内，把两个文件分别解压

进入ArkTradingbot-Beta文件夹中，鼠标右键点击ArkTradingbot.exe属性选择以管理员身份运行

有的小伙伴到了这一步,可能会遇到程序报错，不要担心这是因为您的电脑缺少一个微软粑粑.Net依赖程序将它下载安装即可

使用邮箱/手机号注册

实例中用的是QQ邮箱进行的注册-输入好之后，我们点击图中的领取盲盒按钮（小伙伴们在使用注册时尽量不要选择QQ邮箱，因为有时可能收不到验证信息)

此时我们需要进入邮箱来接收OKX官方发送的注册验证码


复制OKX注册验证码


将验证码输入/复制到下图标记处，并点击完成注册按钮


给你的账户创建一个密码，密码格式最好以大写字母+小写字母+数字+特殊符号
例如：QWZ20230711mosi!

点击图中的确认

选择一个国家，这里你需要注意的是，你属于那个国家的就如实填写便好

再一次点击下一步

我们会正式进入到OKX的官方站点

网页端UID获取

我们来到首页之后，将鼠标移动至右上角小人儿图标处，此时我们就可以看到个人账户的UID了


手机APPUID获取

在手机APP中，我们按照下图流程也能看到我们账户的UID信息

申请API

我们需要在OKX的官网上申请一个V5API用于我们的软件与交易所进行通讯，同时在我们回测的时候通常都需要到商品的历史数据下载以便我们对策略机器人的回测及优化

点击下图中的创建API Key按钮

点击下图中的申请V5API按钮

勾选API交易，填写相应的配置，注意这里只能勾选我所选择的选项

  • 读取
  • 提现
  • 交易

前往你的邮箱并将验证码填写至此处，若你进行了谷歌验证，将你谷歌验证APP中的验证码填写在相当对应的地方

复制图中的验证码，填写在相应的地方

请牢记下图的信息，注意不要将Key信息透露给任何一个人,亲妈都不行~

至此关于券商账户的注册就完成了，接下来我们需要进入到软件注册环节非常简单，2分钟不到就可以完成

Arktradingbot量化软件注册与使用

打开我们提前下载好的Arktradingbot.exe
例：以下是我个人的路径，实际路径请自己决定
文件位置：./（绝对路径）
盘符：D盘 👉 文件夹：./ArkTradingbot/ArkTradingbot-Beta/ArkTradingbot.exe


此时双击打开ArkTradingbot.exe文件，会弹出以下窗口

初次使用量化机器人您需要注册一个账号,请按照以下流程进行 ，至于UID从哪儿获取，大家可以前往 网页端UID获取/手机APPUID获取

注册时，有些小可爱可能不太明白怎么填写注册信息，这里 昵称 账号 密码 邮箱您可以按照个人的喜好进行填写,下面将举例和大家说明

例子：

昵称：股市苍老师
账号：canglaoshi
密码：woaigushi123~
邮箱：canglaoshi@gmail.com

Key如何进行填写

正确的将KEY填写至注册信息表中：
请输入Public Key = API Key
请输入Secret Key = 密钥
请输入Password: = 申请API时填写的密码
注意：大家在填写Key时，还请认真填写，否则会导致量化机器人不能正常运行工作并报错

关于申请API信息，请前往申请API篇幅查看，至此软件的整个注册流程便完成了

小结

  • 注册okx账号
  • 获取账号uid（这很重要，在注册软件账户的时候是必须要填写的内容）
  • 申请V5API（用于量化机器人历史数据的下载/仿真模拟交易时需要与OKX交易所的数据进行交互通讯）
  • 注册量化软件账号（此账号并不是OKX的账号，请大家不要混淆了）
